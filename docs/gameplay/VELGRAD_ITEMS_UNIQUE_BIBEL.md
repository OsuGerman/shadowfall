# VELGRAD — UNIQUE-WAFFEN-BIBEL (50 Stück)

> Jede Waffe ist lore-konsistent zur **Velgrad Lore-Bibel** und referenziert Aspekte, Fraktionen, Ereignisse oder NPCs. Stats sind POE-Style-Modifiers. Drop-Quellen sind klar definiert für deinen Loot-Designer.

**Stat-Konventionen:**
- Implicit = immer aktiv, gehört zum Waffentyp
- Explicit = die unique-Modifier dieser Waffe
- Drop = wo es zu finden ist
- Tier = Leveling (L) / Mid (M) / End (E) / Mythic (X)

## ANHANG — UNIQUE-DESIGN-PRINZIPIEN

1. **Jede Waffe erzählt eine Geschichte.** Wenn der Flavor-Text generisch ist, ist die Waffe schlecht designed.
2. **Mechanik = Lore.** Wenn eine Waffe Stun macht, hat das einen Grund (Kharn schläft). Wenn sie Lärm dämpft, auch (Stille Schritte / Erblinde Kirche).
3. **Trade-Offs.** Mythische Waffen haben oft Nachteile. Tinten-Items reduzieren Resistances. Eid-Items zwingen zu Skill-Reihenfolge. Macht sie interessant statt OP.
4. **Visible-State.** Jede Waffe sollte einen visuellen Hinweis auf ihre Mechanik geben (Atemzähler zeigt Counter, Bischof-Auge schließt sich bei Bösem, Drei-Pagoden glühen bei jedem Hit).
5. **Set-Boni.** Erblinde Kirche / Saatträger / Mahnmal / Speerschwestern / Stille Schritte / Eisenwächter haben Set-Boni mit 2/3/4 Items.
6. **Story-Verzahnung.** 5-10% aller Uniques sind direkt mit NPCs oder Story-Events verknüpft (Vossharils Bruder, Korvens Wahrheit, Tameris' Suchender, Kharns Träne).
7. **Endgame-Mythics.** Mit X markiert. Drop nur in Atlas oder spezifischen Boss-Encountern. Game-Changer.
